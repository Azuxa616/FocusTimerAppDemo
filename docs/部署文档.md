# 部署文档

## 环境要求
- Android Studio Ladybug 或更高版本
- 已安装 Android SDK 35 及以上
- JDK 17

## 获取代码
- 使用 Git 将仓库克隆到本地，或直接下载压缩包解压

## 导入工程
- 打开 Android Studio，选择“Open an Existing Project”
- 指定到 `FocusTimer` 目录
- 等待 Gradle 同步完成

## 构建与运行
- 在工具栏中选择要运行的设备（真机或模拟器，Android 版本不低于 35）
- 选择 `app` 作为运行配置
- 点击“Run”按钮即可编译并安装到设备

## 生成可执行安装包
- 在 Android Studio 菜单中选择 `Build > Generate Signed App Bundle / APK`
- 选择 `APK` 类型
- 选择模块 `app`
- 使用已有密钥库或创建新的密钥库
- 按向导完成签名配置后生成 APK
- 生成的 APK 位于 `app/release` 目录下
